# Agency OS — Demo

Internal agency management system demo.

## Stack
- Pure HTML + CSS + JS (no framework, no build step)
- Works on Vercel as static site

## Roles
- `OPERATIONS_SALES` — full access: projects, tasks, leads, finance, AI analytics
- `PM` — projects, tasks, finance, AI spec generator
- `DEV_WEBFLOW` — Webflow projects & tasks only
- `DEV_WORDPRESS` — WordPress projects & tasks only

## Deploy
Push to GitHub → connect to Vercel → done.

## Local
Just open `index.html` in browser. No server needed.
